# Twi Sentiment Analysis

This project explores sentiment analysis for Ghanaian Twi using pretrained transformer models.
The repository currently provides a command line interface and a simple web demo for analysing sentiment in Twi text.

**Status:** Initial version with model interface and demo in place. Planned extensions include baseline models, quantitative evaluation on Twi datasets, and qualitative error analysis.
